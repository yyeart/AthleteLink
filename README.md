## Краткое описание проекта
AthleteLink — это веб-приложение для поиска и организации любительских спортивных игр. Пользователи могут создавать запросы на набор игроков, указывать вид спорта, место и время проведения, а другие — находить эти игры по фильтрам и подавать заявки на участие в них. Встроенная система рейтинга по каждому виду спорта превращает любительские встречи в понятный «игровой прогресс»: после матчей рейтинг обновляется, что мотивирует участников играть чаще и ответственнее.
Актуальность проекта AthleteLink обусловлена, во-первых, сохраняющимся недостатком регулярной физической активности, что связано с ограниченным количеством удобных инструментов для поиска и организации любительских спортивных мероприятий.Во-вторых, для современной молодежи характерен дефицит очного социального взаимодействия, а потому востребованы сервисы, которые не только помогают находить спортивные активности, но и создают условия для расширения круга общения и формирования устойчивых социальных связей.Система рейтинга, реализованная в нашем проекте добавляет элемент геймификации: пользователи видят свой прогресс и репутацию в разных видах спорта, что стимулирует честную игру и регулярное участие.

## Как запустить проект?
1. Командой git clone -b \<текущая ветка разработки\> https://gitlab.mai.ru/layar/athletelink.git<br>
склонировать репозиторий (на настоящий момент текущая ветка разработки - requests-react)
2. В файл .env, находящийся по пути athletelink/AthleteLink/.env, вставить
```
POSTGRES_DB=athletelink_db
POSTGRES_USER=athletelink_user
POSTGRES_PASSWORD=verysecretpassword
```
3. Введите docker compose up --build
4. Перейдите по адресу localhost:8080 в браузере
